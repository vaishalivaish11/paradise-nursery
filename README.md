# 🌿 Paradise Nursery - React Shopping App

## 📌 Project Description
Paradise Nursery is a React-based shopping application for houseplants. It allows users to browse plants, add them to a shopping cart, update quantities, and view total cost dynamically.

---

## 🚀 Features

### 🏠 Landing Page
- Company name: Paradise Nursery
- Background image
- Short description
- Get Started button → navigates to products page

### 🌿 Product Listing Page
- Minimum 6 houseplants
- Organized into 3 categories (Indoor, Outdoor, Succulents)
- Each plant includes:
  - Image
  - Name
  - Price
  - Add to Cart button

### 🛒 Shopping Cart Page
- Displays all selected plants
- Shows:
  - Total quantity
  - Total price
- Features:
  - Increase quantity
  - Decrease quantity
  - Delete item
- Continue Shopping button
- Checkout button

---
